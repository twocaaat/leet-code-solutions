# 0. Template

***Easy***

This is the **_template_**.
The function gets `x` and returns `x - 1`.

**Example 1:**

> **Input:** 10 \
> **Output:** 9 \
> **Explanation:** 10 - 1 = 9

**Constraints:**

* `-10^9 <= x <= 10^9`
* `x` is number.

---

# Solutions:
## Solution 1:
> ### Runtime
> **0** ms | Beats **100%**

> ### Memory
> **0** MB | Beats **100%**
```csharp
public class Solution
{
    public int MinusOne(int x) => x - 1;
}
```
